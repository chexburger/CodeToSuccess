The Grave Site

As you develop the Grave Site you will learn the basics of coding a website The course will cover many different tags, but no one can remember all of them. If you need to look up or reference a tag or just remember what one does go look around w3schools.com.

Web Development Essentials

In Web Essentials we will learn how to code the basic structure of the page using HTML 5 as well as style the page using CSS.

Web Databases

In Web Databases you will learn how to setup and structure a Database of tomb stones using MySql.

Server Side Scripting

In Server Side Scripting we will be teaching you about PHP. PHP allows you to make a website dynamic, as well as interact wih, retrieve and post data to the Database.

Web Development Essentials

Client Side Scripting will cover JavaScript & JQuery adding additional funcionality to the website and completeing the badic language used in Web Development. By the end of this you will have completed most of the core classes in the program.
